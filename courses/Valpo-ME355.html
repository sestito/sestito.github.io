ME 355 - System Modeling and Numerical Methods

Student Learning Objectives

Students who complete ME 355 should be able to:

  1. Model a mechanical engineering system such as a thermal-fluid system, dynamic system, etc.
  2. Model a bioengineering system such as a musculoskeletal system, tissue system, blood flow, etc.
  3. Model a static system such as a truss, frame, etc.
  4. Model a dynamic system such as thermal conduction, vibration, fluid flow, etc.
  5. Use MATLAB to solve numerical problems
  6. Develop and document computer programs so that others can understand and use them to solve engineering problems.
  7. Select and apply the appropriate numerical method as part of an engineering problem solving methodology.
  8. Apply a numerical iterative method to solve a model
  9. Apply numerical integration to solve a model
  10. Apply an ordinary differential equation solution technique to numerically solve initial value problems consisting of a system of linear ordinary differential equations of any order
  11. Apply an optimization technique to optimize a system represented by a model
  12. Validate numerical results produced from applying a numerical solution technique to a model of a physical system
  13. Interpret numerical results produced from applying a numerical solution technique to a model of a physical system
  14. Describe how similar modeling techniques can be applied to different types of physical systems
  15. Describe how similar numerical solution techniques can be applied to solve models of different types of physical systems
