ME 361 - Mechatronics Laboratory

Student Learning Objectives

At the conclusion of this laboratory, successful students will:

  1. Program Clicks and Allen-Bradly PLCs.
  2. Program Kincko HMI interface to work with Clicks PLCs.
  3. Use National Instruments cRIO FPGA and RT modules for high speed data acquisitions and processing
  4. Use cRIO to operate ABB robots and carry out basic image processing.
  5. Integrate potentiometers, proximity sensors, hall effect sensors, optical sensors, and
  6. Drive stepper and servo motor systems using cRIO.
  7. Program Advanced-Motion motor drivers for automation of a gantry system
  8. Interface with pneumatic valves and pneumatic actuators using PLC.
